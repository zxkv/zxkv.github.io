#### 辅助软件

- **ONE-KEY-HiDPI**：[https://github.com/xzhih/one-key-hidpi](https://github.com/xzhih/one-key-hidpi)「✅」

    - 「🔥」一键开启 MacOS HiDPI，适配屏幕显示器，优化显示器显示模糊问题
    - 「💻」MacOS

- **Motrix**：[https://motrix.app/zh-CN](https://motrix.app/zh-CN)「✅」

    - 「🔥」一款全能的下载工具(支持下载 HTTP、FTP、BT、磁力链接等资源)
    - 「💻」MacOS、Windows、Linux

    **IMFILE**：[https://motrix.app/zh-CN](https://motrix.app/zh-CN)「✅」

    - 「🔥」一款全能的下载工具(支持下载 HTTP、FTP、BT、磁力链接等资源)
    - 「💻」MacOS、Windows、Linux

- **Gopeed**：[https://imfile.io](https://imfile.io)「✅」

    - 「🔥」顺畅、高速、强大而便捷的下载工具(支持下载 HTTP、FTP、BitTorrent、Magnet 等)
    - 「💻」MacOS、Windows、Android

- **AB Download Manager**：[https://abdownloadmanager.com](https://abdownloadmanager.com/)「❌」

    - 「🔥」开源下载器(支持浏览器插件和客户端安装版)
    - 「💻」Windows、Linux

::: danger 注意事项：

- [**one-key-hidpi**](https://github.com/xzhih/one-key-hidpi/blob/master/README-zh.md)**：MacOS** 的 **DPI** 机制和 **Windowsdows** 下不一样，比如 **1080p** 的屏幕在 **Windowsdows** 下有 125%、150% 这样的缩放选项，而同样的屏幕在 **MacOS** 下，缩放选项里只是单纯的调节分辨率，这就使得在默认分辨率下字体和 UI 看起来很小，降低分辨率又显得模糊。通过此工具，可以一键开启 **MacOS** **HiDPI**，适配屏幕显示器，优化显示器显示模糊问题。

:::
