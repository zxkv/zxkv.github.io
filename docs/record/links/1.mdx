# 前端收藏

### ✂️ 截图工具

- [Snipaste](https://zh.snipaste.com/)「✅」

    - MacOS、Windows、Linux

- [PixPin](https://pixpin.cn/)「🔥、✅、💰」

    - MacOS、Windows
    - 支持录屏、复制内容、导出指定格式图片、视频等

- [FastStone Capture](https://www.faststonecapture.cn/)「💰」

    - Windows

- [Xnip](https://zh.xnipapp.com/)

    - MacOS

- [PicPick](https://picpick.app/zh/)「💰」

    - MacOS、Windows

- [Lightshot](https://app.prntscr.com/zh-cn/index.html)

    - MacOS、Windows

- [ShareX](https://getsharex.com/)
    - MacOS、Windows

### 文档工具

- [Typora](https://typora.io/)「✅」
- [MWeb](https://zh.mweb.im/)
- [幕布](https://mubu.com/)
