# 编辑器配置 - Trae

### IDE 简介

&emsp;&emsp;`Trae` 字节跳动开源的 Trae IDE，旨在提供一个强大的云原生 IDE 平台，让开发者能够更便捷地进行开发、测试和部署。由于众所周知的原因，国内版和国外版在某些方面存在差异。

&emsp;&emsp;`Trae` 同 `Windsurf Editor`一样，也可以直接采用 **vscode** 的配置，按照 **vscode** 的需求配置安装插件即可，无需额外安装，前提需要注册账号才可以使用编辑器。默认安装插件的源受网络限制，下载和搜索插件比较慢，如果要求高，需要下载插件到本地进行安装，或者自行配置代理来加速插件的下载和安装。

### 对比分析

| 特性     | 国内版                         | 国外版                         |
| -------- | ------------------------------ | ------------------------------ |
| 功能     | 核心功能一致，可能针对国内优化 | 核心功能一致，可能针对国外优化 |
| 社区     | 国内开发者社区为主             | 全球开发者社区为主             |
| 数据安全 | 符合中国法律法规               | 符合当地法律法规               |
| 访问速度 | 国内访问更快                   | 国外访问更快                   |
| 用户体验 | 默认中文，更符合国内用户习惯   | 默认英文，更符合国外用户习惯   |
