# 编辑器配置 - Trae

### IDE 简介

&emsp;&emsp;`Trae` Trae is an adaptive AI IDE that transforms how you work, collaborating with you to run faster.

&emsp;&emsp;`Trae` 同 `Windsurf Editor`一样，也可以直接采用 **vscode** 的配置，按照 **vscode** 的需求配置安装插件即可，无需额外安装，前提需要注册账号才可以使用编辑器。默认安装插件的源受网络限制，下载和搜索插件比较慢，如果要求高，需要下载插件到本地进行安装，或者自行配置代理来加速插件的下载和安装。

### IDE 下载

- [Trae 官网](https://www.trae.ai)「👈 点击查看下载」
- 支持系统
    - MacOS
        - Apple Silicon
        - Intel
    - Windows[开发中]
